## 1. 项目概述

### 1.1 项目目标
打造一个面向大陆用户的免费算命网站，通过观看视频广告解锁高级功能，实现流量变现。

### 1.2 目标用户
- 年龄：25-55岁
- 地域：主要大陆用户，兼顾香港用户
- 需求：八字命理、运势查询、抽签问卦等传统命理服务

### 1.3 商业模式
- 基础功能免费，但需观看15-30秒视频广告
- 广告收入来源：Google AdSense / 穿山甲 / 腾讯广告
- 服务器：GitHub Pages（美国）+ Cloudflare CDN

## 3. 功能模块

### 3.1 核心功能（完整版）

#### 3.1.1 八字排盘
- **输入**: 姓名、出生年月日时分、性别
- **输出**: 
  - 年柱、月柱、日柱、时柱
  - 十神关系
  - 五行强弱分析
  - 大运排盘
- **广告**: 查看完整结果前观看视频

#### 3.1.2 命盘解读
- **输入**: 八字排盘结果
- **输出**:
  - 性格分析
  - 事业运势
  - 感情运势
  - 财运分析
  - 健康建议
- **广告**: 查看每个板块前观看视频

#### 3.1.3 生肖运势
- **输入**: 生肖选择
- **输出**:
  - 今日运势
  - 本周运势
  - 本月运势
  - 今年运势
- **广告**: 查看详细运势前观看视频

#### 3.1.4 抽签求卦
- **输入**: 问题描述（可选）
- **功能**:
  - 观音灵签
  - 妈祖灵签
  - 关帝灵签
  - 月老姻缘签
- **广告**: 每次抽签前观看视频

#### 3.1.5 解梦查询
- **输入**: 梦境关键词
- **输出**: 梦境寓意、凶吉判断、心理学解读
- **广告**: 查看完整解读前观看视频

#### 3.1.6 姓名测试
- **输入**: 姓名（中文）
- **输出**: 五行分析、卦象吉凶、名字评分
- **广告**: 查看详细分析前观看视频

#### 3.1.7 星座查询
- **输入**: 星座+生日
- **输出**: 今日/本周/本月运势、性格分析、配对指数
- **广告**: 查看详细内容前观看视频

#### 3.1.8 风水罗盘
- **功能**: 家居风水、办公室风水、颜色风水
- **输出**: 风水建议、布局调整方案
- **广告**: 查看风水方案前观看视频

#### 3.1.9 黄历查询
- **输出**: 今日宜忌、吉时冲煞、神煞方位
- **广告**: 查看详细信息前观看视频

#### 3.1.10 手相/面相
- **输入**: 上传图片或手动输入特征
- **输出**: 掌纹解读、面相分析
- **广告**: 查看分析结果前观看视频

## 7. 风险与注意事项

### 7.1 合规风险
- 算命内容属于文化类，不涉及封建迷信定义边缘
- 避免使用"预测未来"、"保证准确"等绝对化用语
- 加入免责声明："仅供娱乐参考"

### 7.2 技术风险
- GitHub Pages 在大陆访问可能不稳定 → 使用 Cloudflare
- 激励视频广告大陆平台审核严格 → 准备多个广告平台备选

### 7.3 内容风险
- 签文数据需确保来源合规
- 解梦内容需无明显迷信表述
- 定期检查敏感词
